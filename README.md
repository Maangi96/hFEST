# Initial steps

create a repo from github

1. git init
2. git remote add origin <repo url>
   - git remote -v
3. git add .
4. git commit -am 'initial'
5. git push -u origin main

1. git add .
2. git commit -am 'what you have done'
3. git push -u origin main
